# 本地旅游攻略

一个展示本地旅游景点的趣味展示网站，帮助用户发现身边的美景和有趣的地方。

## 项目介绍

本项目是一个"趣味主题展示站"，以"本地旅游攻略"为主题，收录了12个各具特色的本地旅游景点，涵盖自然风光、历史文化、休闲娱乐、户外运动、美食购物等多种类型。

## 功能特点

### 第一节课完成的功能
- 响应式布局，支持手机端和电脑端浏览
- 旅游景点卡片，每个卡片包含图片、分类标签、标题、描述、位置和评分
- 美观的卡片悬停动画效果
- 图片懒加载功能，优化页面加载速度
- 平滑的滚动动画和过渡效果
- 返回顶部按钮

### 第二节课完成的功能
- ✅ **搜索功能**：实时搜索景点名称
- ✅ **分类筛选**：按类别筛选景点（自然风光、历史文化、休闲娱乐、户外运动、美食购物）
- ✅ **排序功能**：支持按评分高低、名称字母顺序排序
- ✅ **点赞功能**：每个景点可点赞，数据保存到localStorage
- ✅ **分享功能**：支持复制分享链接或调用系统分享API
- ✅ **Lightbox画廊**：点击图片放大查看，支持左右切换和键盘操作
- ✅ **滚动动效**：卡片滚动进入视口时的淡入动画
- ✅ **悬浮交互**：卡片悬停时的缩放和上浮效果

### 第三节课完成的功能
- ✅ **个人简历页面**（about.html）：展示站长信息、旅行经历、技能和联系方式
- ✅ **内容扩充**：景点数量从8个增加到12个
- ✅ **留言板功能**：支持用户留言，数据保存到localStorage
- ✅ **评分功能**：星级评分弹窗，用户可对景点进行评分
- ✅ **完善移动端适配**：优化手机端显示效果

## 技术栈

- HTML5
- CSS3（Flexbox、Grid、动画）
- JavaScript（原生 ES6+）
- Unsplash 免费图片
- localStorage 数据持久化

## 使用爬虫获取图片

如需更新或添加更多图片，可以运行爬虫脚本：

```bash
cd scraper
pip install -r requirements.txt
python scraper.py
```

爬取的图片将保存到 `images` 目录。
